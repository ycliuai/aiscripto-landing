# AiScripto 登錄網站 - 架構文檔

## 📋 概述

這是AiScripto的完整展示登錄網站，設計用於向資金來源、投資者和潛在客戶展示產品的多樣性、AI創新性和商業價值。

## 🌐 網站結構

### 文件組織

```
aiscripto-landing/
├── index.html              # 首頁 - 產品概述和核心價值主張
├── features.html           # 功能特性頁面 - 詳細的產品功能介紹
├── workflow.html           # 工作流程頁面 - 完整的轉換管道展示
├── architecture.html       # 技術架構頁面 - 系統設計和技術棧
├── use-cases.html          # 使用案例頁面 - 實際應用和ROI分析
├── getting-started.html    # 快速開始頁面 - 安裝和入門指南
├── contact.html            # 聯絡頁面 - 聯繫方式和表單
│
├── styles.css              # 全局共享樣式表
├── CNAME                   # GitHub Pages自定義域名配置
├── README.md               # 項目說明
├── WEBSITE_STRUCTURE.md    # 本文件
│
└── docs/                   # 文檔資料夾（來自主項目）
    ├── getting-started-guide.md
    ├── test-case-authoring-guide.md
    └── architecture_spec/
        ├── AiScripto_ARCHITECTURE_EN.md
        └── AiScripto_ARCHITECTURE_ZH_TRAD.md
```

## 🎨 設計系統

### 色彩方案
- **主色深灰：** #1a1a1a（導航、標題）
- **次色淡灰：** #2d2d2d（漸變背景）
- **強調色橙：** #ff9900（按鈕、邊框、重點）
- **強調色藍：** #0066cc（次要按鈕、鏈接）
- **文字深色：** #1a1a1a（正文）
- **文字淺色：** #666（輔助文本）
- **背景白：** #ffffff（卡片、區段）
- **背景淺灰：** #f9f9f9（整體背景）

### 排版
- **字體家族：** -apple-system, BlinkMacSystemFont, 'Segoe UI', 'Helvetica Neue', sans-serif
- **H1：** 48px 700 weight
- **H2：** 32px 600 weight（帶橙色下邊框）
- **H3：** 20px 600 weight
- **正文：** 14px-16px，line-height 1.6-1.8

### 響應式斷點
- **桌面：** 1200px (max-width: 1200px)
- **平板：** 768px (max-width: 768px)
- **手機：** 480px (max-width: 480px)
